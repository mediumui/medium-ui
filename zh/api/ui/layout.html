    提示

    需要展示页脚固定栏时，使用slot="footer"。当页脚为空时，内容区域会自动填充页脚空间

    参数

    参数 说明 类型 可选值 默认值
    title 页头标题 String - -
    loading 加载中效果，当loading为true时内容区域会出现加载效果 Boolean true false

    Slot

    name 说明
    - 内容区域
    head 页头自定义，注意该自定义为Layout右上角，不会影响页头的返回和标题
    footer 页脚自定义，当没有使用该slot时，内容区域会往下填充
