    me-table组件兼容el-table所有的props和event，不支持调用el-table中的methods

    me-table-column继承el-table-column,即在el-table-column基础上进行了扩展，兼容所有特性和api。

    参数

    参数 说明 类型 可选值 默认值
    - 原el-table属性，为了处理数据不支持配置data Any - -
    page 是否使用底部分页功能 Boolean true false
    pageSizes 分页可选页码 Array - [10, 20, 30, 50, 100]
    request 数据请求响应，异步回调 Function - -

    事件

    事件名称 说明 回调参数
    - 原el-table事件 -

    方法

    方法名 说明 参数
    refresh 刷新数据 Function(resetPage: Boolean)

    me-table-column参数

    参数 说明 类型 可选值 默认值
    - 原el-table-column属性 Any - -
    search 是否搜索 Boolean true false
    se- me-search-item属性，search为true时生效 Any - -
